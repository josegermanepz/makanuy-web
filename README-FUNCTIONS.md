# Funciones independientes de Makanuy

## Variables y bindings de Cloudflare Pages

- D1 binding: `DB`
- Secret: `MP_ACCESS_TOKEN`
- Variable: `MP_PUBLIC_KEY`
- Secret opcional para email: `RESEND_API_KEY`
- Variable: `EMAIL_FROM`
- Variable: `BUSINESS_EMAIL`

Ejecutar `schema.sql` en la base D1 antes de probar reservas.
