Datos que solicitamos

Para citas podemos solicitar nombre, correo, teléfono, servicio, fecha y horario. Para empresas también podemos solicitar organización, número aproximado de colaboradores, modalidad y mensaje. No envíes diagnósticos, estudios ni información clínica sensible mediante formularios abiertos.

  • Responder solicitudes y brindar información sobre servicios.
  • Gestionar disponibilidad, confirmación, cancelación y reprogramación de citas.
  • Dar seguimiento a solicitudes empresariales.
  • Prevenir abuso y mantener la seguridad del sitio.

Proveedores tecnológicos

Utilizamos Cloudflare para alojar el sitio, su base de datos y funciones de seguridad; Google Calendar para consultar disponibilidad; y, cuando está configurado, un proveedor de correo transaccional para enviar avisos. Estos proveedores procesan únicamente la información necesaria para prestar sus servicios.

Conservación y seguridad

La información se conserva durante el tiempo necesario para atender la solicitud y cumplir obligaciones aplicables. El acceso se limita a Makanuy y a los proveedores necesarios para operar el servicio.
